Welcome to the University of Reading Atmospheric Observatory database.

This website allows measurement data to be downloaded from the University of Reading's Atmospheric Observatory (URAO). Data holdings are updated at least once per day with the latest values, and data can be downloaded in a variety of formats.

Available data

A list of the datasets currently available can be viewed on the "View datasets" page. This describes the datasets and the meteorological variables therein.

Getting access

An access token must be requested in order to download URAO data. Please contact Roger Brugge for more information. When requesting access, please include the dataset(s), variable(s) and time period(s) of interest.

Downloading data

Once you have an access token, data can be downloaded using the "Get Data" link on the menu (left)

Data can be downloaded in a choice of formats:

  • CSV: Comma-separated variables: Suitable for Microsoft Excel and data analysis programs)
  • JSON: Javascript Object Notation: Suitable for ingestion by computer software
  • HTML: A table of values best viewed in a Web Browser

Missing data

On some dates there may be no measurement data at all. If data is missing for the whole day, then this day will be completely skipped in the downloaded data to save space. This is reflected by a gap in the timestamps.

If part of the data are missing on a certain day, missing value markers are substituted to ensure regular data. The missing value marker is chosen in the "Download options" section of the form.
